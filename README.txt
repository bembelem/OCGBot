Новостной бот ДВФУ
Этот проект представляет собой Telegram-бота, предназначенного для получения новостей с сайта Дальневосточного федерального университета (ДВФУ). Бот парсит новости с официального сайта университета и отправляет их пользователям в виде сообщений с заголовками, текстом новости и ссылкой на источник.
Установка:
1)Клонируйте этот репозиторий на свой локальный компьютер:
В IDE нажмите на "Get From VCS" и укажите ссылку на репозиторий, https://github.com/betiebembelem/OCGBot
2)Установите необходимые зависимости с помощью pip(если они не установились)
pip install -r requirements.txt
3)Создайте файл keys.py в корневой директории проекта и добавьте в него свой API-ключ Telegram-бота:
API = 'your_telegram_bot_api_key'

Использование:
1)Запустите бота
2)В Telegram найдите своего бота и отправьте ему команду /start. Бот отправит вам приветственное сообщение и клавиатуру с доступными опциями.
3)Используйте клавиатуру для взаимодействия с ботом:

Нажмите "Последняя новость", чтобы получить последнюю новость с сайта ДВФУ.
Нажмите "Следующая новость" или "Предыдущая новость" для навигации по новостям.
Нажмите "Сайт ДВФУ", чтобы получить ссылку на официальный сайт университета.
Нажмите "Описание бота", чтобы получить информацию о боте.
